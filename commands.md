# Git Commands Reference

1. **git config --global user.name "Your Name"**  
   Set your Git username for commits.

2. **git config --global user.email "youremail@example.com"**  
   Set your Git email for commits.

3. **git init**  
   Initialize a new Git repository in your current folder.

4. **git clone [repository-url]**  
   Clone an existing remote repository to your local machine.

5. **git status**  
   Check the status of files: which are staged, unstaged, or untracked.

6. **git add [file]**  
   Stage a specific file for commit.

7. **git add .**  
   Stage all changes in the current directory for commit.

8. **git commit -m "message"**  
   Commit staged changes with a descriptive message.

9. **git log**  
   View the commit history of the repository.

10. **git push origin [branch-name]**  
    Push your local commits to a remote repository on the specified branch.
